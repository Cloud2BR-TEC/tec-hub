# About Cloud2BR TEC

Cloud2BR TEC is a technology learning and enablement platform focused on helping individuals, teams, and organizations explore, learn, and apply modern technology with confidence.

## What TEC provides

| Experience | Purpose |
| --- | --- |
| Self-paced learning paths | Build practical knowledge progressively, from fundamentals through production practices |
| Guided implementation content | Connect concepts to architecture, tools, and working patterns |
| Practical labs and resources | Create space to experiment, apply, and adapt learning |
| Community resources | Share learning materials and collaborate through open repositories |

## Focus areas

- Cloud fundamentals and architecture
- Data and AI foundations
- DevOps and platform engineering
- Security and governance practices
- Productivity with modern developer tools

> **TEC mission:** Enable people and organizations to explore, learn, and apply technology with confidence.

!!! tip "Use the hub as your starting point"
    Each catalog entry links to its GitHub source and, when available, its published learning site. Read the 101 path first when entering a topic, then follow the matching 102 and 103 paths as your needs become more practical and operational.
